Pill badge for status or classification — never decorative. Tinted by semantic tone.

```jsx
<Badge tone="success" dot>정상</Badge>
<Badge tone="accent">MCP</Badge>
<Badge tone="info">REST</Badge>
<Badge>GET</Badge>
```

Tones: `neutral` / `accent` / `info` / `success` / `warning` / `danger`. `dot` adds a leading status dot.
