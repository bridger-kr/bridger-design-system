Surface container for repeated items, modals, and framed tools. Not a decorative wrapper.

```jsx
<Card variant="panel" interactive padding={20}>…</Card>
<Card variant="muted">inset well</Card>
```

Variants: `default` (surface + subtle shadow), `muted` (sunken well), `raised` (elevated), `panel` (flat console panel). `interactive` strengthens the border + lifts the surface on hover.
