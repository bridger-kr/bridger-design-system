The Bridger wordmark — mark-only (the weak bridge symbol is retired). The brand detail is a single persimmon period, flex.team-style. Set in Pretendard heavy, tight tracking.

```jsx
<BrandLogo size="md" />
<BrandLogo size="lg" period={false} />
```

Sizes `sm` / `md` / `lg` / `xl` or an explicit px number. `period` toggles the persimmon dot.
